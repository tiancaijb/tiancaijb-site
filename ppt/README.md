# 幻灯片素材

瑞士国际主义风格（Swiss Style），来自歸藏 guizang-ppt-skill。

## 文件

| 文件 | 用途 |
|------|------|
| `assets/template-swiss.html` | 瑞士风格模板（完整可运行） |
| `references/layouts-swiss.md` | 22 种瑞士风格布局 S01-S22 |
| `references/themes-swiss.md` | 4 套主题色：IKB蓝/柠檬黄/柠檬绿/安全橙 |
| `references/swiss-layout-lock.md` | 瑞士风格锁定规则 |
| `validate-swiss-deck.mjs` | 校验脚本 |

## 使用

1. 拷贝 `assets/template-swiss.html` 到工作目录
2. 选主题色（`references/themes-swiss.md`）
3. 选布局（`references/layouts-swiss.md`）
4. 填充内容到 `<!-- SLIDES_HERE -->` 位置
